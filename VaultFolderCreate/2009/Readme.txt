Overview:
VaultFileBrwoser is a tool that allows users to perform common Vault activities such as uploading and downloading files.  It also displays BOM and association file data.
 
 
VaultFileBrwoser contains the following features/concepts:
	- Log in/security
	- Browsing folders and files
	- Uploading files
	- Downloading files
	- File BOM
	- File associations
	- File Search
	- Error handling


To Use:
If you don't have WSE 3.0, download and install it from Microsoft.  When you install, choose the Visual Stuido integration option. http://msdn.microsoft.com/webservices/building/wse/default.aspx

Next open VaultList.sln in Visual Studio.  The project should open with no errors.
